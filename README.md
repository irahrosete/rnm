# Rick and Morty Wiki
https://rickmortywiki.vercel.app/

Learn about the characters, episodes and locations of the critically-acclaimed adult-animated, science-fiction sitcom, [Rick and Morty](https://en.wikipedia.org/wiki/Rick_and_Morty).

### Features

- view by:
    - character
    - episode
    - location
- search
- filter
- pagination

### Stack

- React

### Tools

- Bootstrap

### Deployment

- Vercel
